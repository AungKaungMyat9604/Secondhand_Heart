Project Title: Secondhand Heart (Toys Auction) 

Introduction
Online marketplaces have become increasingly popular for buying and selling used items, providing a convenient platform for people to exchange goods without physical limitations. Among collectors and hobbyists, secondhand toys and action figures hold significant value due to rarity, nostalgia, and affordability. However, finding reliable platforms specifically designed for auctioning such collectibles can be difficult.
The goal of this project is to develop a Web-Based Auction System for Secondhand Toys and Figures that allows users to buy and sell collectible toys through an online bidding process. The system will enable sellers to list their items, while buyers can place bids and compete with others to purchase the items. The platform will provide a secure and user-friendly environment where users can easily browse products, participate in auctions, and track bidding activities.
The system will include an interactive web interface for users and a backend administration panel for managing auctions, users, and product listings. Modern web technologies will be used to ensure smooth functionality, real-time updates, and secure transactions. The main objective of this project is to demonstrate how an online auction platform can simplify the buying and selling process for secondhand collectibles while enhancing the user experience through modern web development tools.


Objectives
• Develop a web-based auction platform for buying and selling secondhand toys and collectible figures.
• Allow users to create accounts and participate in auctions by placing bids on items.
• Enable sellers to list toys and figures with descriptions, images, and starting bid prices.
• Provide a real-time bidding system that automatically updates the highest bid.
• Offer a simple and user-friendly interface for browsing and bidding on items.
• Allow administrators to manage users, products, and auction activities.
• Demonstrate the integration of modern web technologies in building an online auction system.

Key Features
User Features
• Register and log in to the auction platform.
• Browse available secondhand toys and figures listed for auction.
• View detailed information about each item including images, descriptions, and current bid price.
• Place bids on items during the auction period.
• Receive notifications or updates when outbid or when auctions end.
• Track bidding history and personal auction activities.
• Manage personal profile and view items they have listed or bid on.

Seller Features
• List secondhand toys or collectible figures for auction.
• Upload images and provide item descriptions and starting prices.
• Set auction duration and bidding rules.
• Track bids placed on their listed items.
• View auction results once the bidding period ends.

Admin Features (Backend)
• Manage user accounts and seller activities.
• Approve, edit, or remove auction listings.
• Monitor bidding activities and system transactions.
• Handle reports of inappropriate listings or suspicious activity.
• Manage system settings and ensure secure authentication for administrators.

Technology Stack
• Frontend: HTML, CSS, JavaScript, React.js or Bootstrap for responsive UI
• Backend: PHP (Laravel Framework) or Node.js
• API Communication: RESTful APIs (GET, POST, PUT, DELETE)
• Database: MySQL database for storing user accounts, auction listings, bids, and transaction history
• Authentication: Secure login and user authentication system
• Hosting/Deployment: Apache Server, XAMPP, or cloud hosting platforms
• Additional Tools: Git/GitHub for version control
